# backend-cli
# Create Nest App CLI

A simple CLI tool to quickly scaffold a **NestJS backend project** with optional database and ORM setup.

## Features
* Creates a NestJS project automatically
* Installs database drivers
* Supports multiple ORMs
* Automates common backend setup tasks

## Tech Stack
* Node.js

## Future Improvements
* Interactive CLI prompts
* Authentication module generation
* Swagger setup
* Environment file generation
